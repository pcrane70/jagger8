Detailed Description

Schedules queries across endpoints one by one.

Author
Mairbek Khadikov
Details:

. "One by one" algorithm - for input endpoints [e1, e2] and queries [q1, q2, q3] executes actions in following order:
(e1, q1), (e2, q1), (e1, q2), (e2, q2), (e1, q3), (e2, q3).

To use this distributor add query-distributor element with type query-distributor-one-by-one in scenario block.

Parameters
<Q>Query type
<E>Endpoint type
